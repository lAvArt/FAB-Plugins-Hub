# How to Report Issues & Request Features

We appreciate your feedback! To help us resolve your issue or implement your idea as quickly as possible, please follow the guidelines below.

## ðŸ› Reporting a Bug

If you believe you've found a bug in one of our plugins, please open a **New Issue** on GitHub and include the following details:

### 1. Title
A clear, concise description of the problem.
*   *Bad:* "Plugin crashes"
*   *Good:* "Crash when using Area Tool on specific Static Mesh"

### 2. Environment
*   **Plugin Version:** (e.g., v1.0) - *Check the Dashboard footer or `.uplugin` file.*
*   **Unreal Engine Version:** (e.g., 5.3.2)
*   **Platform:** (e.g., Windows 11)

### 3. Description
Explain what happened, what you expected to happen, and what actually occurred.

### 4. Steps to Reproduce (Important!)
List of steps to make the bug happen again.
1.  Open level 'X'
2.  Select 'Distance Tool'
3.  Click on 'Actor Y'
4.  ...

### 5. Logs & Screenshots
*   Attach any relevant **Output Logs** (Window > Output Log). Copy/paste the red/yellow lines.
*   **Screenshots** or short **Videos** are incredibly helpful!

## ðŸ’¡ Requesting a Feature

Have an idea to make the tools better? We'd love to hear it!

### 1. Title
A summary of your idea.
*   *Example:* "Add ability to export measurements to PDF"

### 2. Description
*   What problem does this solve?
*   How should it work?
*   Any UI/UX preferences?

### 3. Alternatives
Have you considered other ways to solve this? Why is this feature the best approach?
