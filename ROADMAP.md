# Roadmap

This document outlines the planned features, improvements, and updates across all FAB plugins.

## Overview

This roadmap provides transparency into our development priorities and planned features. It is subject to change based on user feedback, technical considerations, and business needs.

**Note**: Timelines are approximate and may shift. Features may be added, modified, or removed based on development progress and community feedback.

## How to Use This Roadmap

- **In Progress**: Currently under active development
- **Planned**: Scheduled for upcoming releases
- **Under Consideration**: Being evaluated for future development
- **Completed**: Recently finished and released

## General Improvements

### In Progress
- Expanding documentation and tutorials
- Improving compatibility information
- Enhancing issue tracking and response

### Planned
- Video tutorials and guides
- Sample projects and examples
- Community contribution guidelines
- Plugin-specific roadmap details

## Plugin-Specific Roadmaps

Detailed roadmaps for individual plugins will be maintained in their respective documentation directories.

## Feature Requests

We welcome feature requests from the community!

To submit a feature request:
1. Check existing [Issues](https://github.com/lAvArt/FAB-Plugins-Hub/issues) to see if it's already been suggested
2. Create a new issue using the feature request template
3. Provide detailed information about the feature and its use case
4. Engage in discussions about implementation

Popular feature requests with strong community support will be prioritized for development.

## Release Schedule

Release schedules vary by plugin based on:
- Development complexity
- Testing requirements
- Unreal Engine release cycles
- Community feedback and priorities

Major updates typically follow Unreal Engine version releases to ensure compatibility.

## Contributing

While plugin source code is private, you can contribute to this repository by:
- Improving documentation
- Reporting issues and bugs
- Suggesting features and enhancements
- Helping other users in discussions

See [CONTRIBUTING.md](CONTRIBUTING.md) for more information.
