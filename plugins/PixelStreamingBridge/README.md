# Pixel Streaming Bridge (PSB)

Pixel Streaming Bridge (PSB) is an Unreal Engine plugin designed for JSON-based bidirectional messaging between Unreal runtime and web clients.

## Documentation

- [User Guide](./docs/USER_GUIDE.md) - Quick start and basic usage (Blueprints & Web).
- [Web UI Guide](./docs/WEB_UI_DOCS.md) - Using the provided Next.js templates.
- [Technical Details & FAQ](./docs/FAB_PRODUCT_DOCS.md) - Specs, architecture, and common questions.
- [Architecture](./docs/architecture.md) - High-level data flow and component breakdown.

## Features
- Pure JSON messaging contract.
- Provider agnostic (Cirrus, Arcware, Eagle 3D, etc.).
- Robust Unreal Subsystem (GameInstance).
- Reference React/Next.js UI components.
