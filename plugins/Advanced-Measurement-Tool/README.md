# Advanced Measurement Tool (AMT)

**Version:** 1.0  
**Unreal Engine Support:** 5.0+

A professional-grade measurement and analysis plugin for Unreal Editor, designed for level designers, architects, and QA teams.

## Features

*   **Distance**: Measure linear distance, including ground projection and slope angles.
*   **Angle**: Measure angles between three points.
*   **Area**: Calculate the area of polygon selections.
*   **Volume**: Measure bounds and geometric volume of actors.
*   **Snapping**: Comprehensive snapping options including Grid, Bounding Box, and **Mesh Vertex** snapping.
*   **Measurement History**: Keep track of previous measurements with isolation and export capabilities (CSV/JSON).

## Installation

1.  Download the latest release.
2.  Extract to your project's `Plugins` folder.
3.  Enable "Advanced Measurement Tool" in `Edit > Plugins`.

## Usage

*   **Distance Tool (D)**: Click two points to measure. Hold `Shift` to drag the second point.
*   **Angle Tool (A)**: Click three points (Apex, Start, End).
*   **Area Tool (R)**: Click multiple points to define a polygon. Press `Enter` to close.
*   **Volume Tool (V)**: Select an actor to measure its bounding box or geometry volume.
