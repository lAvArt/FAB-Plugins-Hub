# Pixel Streaming Bridge (PSB)

Pixel Streaming Bridge (PSB) is an Unreal Engine plugin designed for JSON-based bidirectional messaging between Unreal runtime and web clients.

## Documentation

- [User Guide](./USER_GUIDE.md) - Quick start and basic usage (Blueprints & Web).
- [Technical Details & FAQ](./FAB_PRODUCT_DOCS.md) - Specs, architecture, and common questions.
- [Architecture](./architecture.md) - High-level data flow and component breakdown.

## Features
- Pure JSON messaging contract.
- Provider agnostic (Cirrus, Arcware, Eagle 3D, etc.).
- Robust Unreal Subsystem (GameInstance).
- Reference React/Next.js UI components.
