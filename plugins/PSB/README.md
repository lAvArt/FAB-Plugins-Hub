# Pixel Streaming Bridge (PSB)

Pixel Streaming Bridge is an Unreal Engine plugin designed to streamline communication between the Pixel Streaming frontend and the Unreal Engine backend.

## Features

- Simplified messaging system.
- Generic payload support.
- Bridge for web UI integration.

## Folder Structure

- `Source`: C++ implementation.
- `Content`: Blueprint and asset resources.
- `Resources`: Icons and metadata.
- `Docs`: Documentation.
