# Arduino Cloud Bridge

**The bridge between Unreal Engine and the Arduino IoT Cloud.**

Seamlessly connect your Unreal Engine projects to physical devices. This plugin provides a high-performance, Blueprint-friendly interface to the Arduino Cloud API, allowing you to read and write device properties in real-time.

*   **Native Integration**: Built specifically for Unreal Engine 5.3+ using the Arduino Cloud OAuth2 API.
*   **Blueprint Ready**: Full support for Blueprints with specialized nodes for polling and updating device values.
*   **Real-time Synchronization**: Configurable poll rates to keep your digital twins in sync with the physical world.
*   **Secure Credential Management**: Encrypted storage of API keys within the Editor's user settings.
*   **Cross-Platform Support**: Optimized for Windows (Win64) with cross-platform compatibility in progress.
