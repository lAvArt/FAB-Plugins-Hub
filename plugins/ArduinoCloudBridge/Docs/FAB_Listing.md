# FAB Listing Information - Arduino Cloud Bridge

## Technical Details

**Features**:
- Connects Unreal Engine to Arduino IoT Cloud via REST API.
- Supports OAuth2 authentication flow.
- Real-time property synchronization (Read/Write).
- Built-in Editor management window.
- Lightweight and performance-oriented.

**Code Modules**:
- `ArduinoCloudBridgeRuntime` [Runtime]
- `ArduinoCloudBridgeEditor` [Editor]

**Number of Blueprints**: 0 (C++ Base)
**Number of C++ Classes**: 12
**Network Replicated**: No (Client-side API)
**Supported Development Platforms**: Windows
**Supported Target Build Platforms**: Windows (Win64)

## Marketing Description

Bring the physical world into your Unreal Engine projects with the **Arduino Cloud Bridge**. Whether you are building smart home simulations, industrial digital twins, or interactive art installations, this plugin provides the most reliable way to bridge the gap between Arduino's ecosystem and Unreal Engine's visuals.

With a Blueprint-first approach, developers can focus on gameplay and logic while the plugin handles the complexities of OAuth2, token refreshing, and JSON parsing.
