# Changelog - Arduino Cloud Bridge

All notable changes to this plugin will be documented in this file.

## [1.0] - 2026-01-30
### Added
- Initial release of the Arduino Cloud Bridge plugin.
- Direct integration with Arduino IoT Cloud API.
- Support for Boolean, Integer, Float, and String properties.
- Blueprint-first runtime subsystem for easy integration.
- Editor window for credential management and device testing.
- Optimized polling system with configurable rates.
