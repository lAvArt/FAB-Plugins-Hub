# Announcements

This directory contains official announcements and updates about FAB plugins.

## Latest Announcements

Check this directory for:
- Major plugin releases
- Important updates and changes
- Upcoming features and plans
- Community updates
- Event announcements

## Staying Updated

To receive announcements:
- **Watch this repository** on GitHub to get notifications
- Check this directory regularly for new announcements
- Review the [Roadmap](../ROADMAP.md) for planned features
- Follow release notes in the [Releases](../releases/README.md) directory

## Announcement Format

Announcements are organized by date in the format `YYYY-MM-DD-title.md` for easy chronological browsing.

## Communication Channels

- **GitHub Repository**: Primary source for documentation and updates
- **Issues**: Bug reports, feature requests, and support
- **Discussions**: Community discussions and Q&A (if enabled)

## Important Notes

- This repository does not contain plugin source code
- All plugins are distributed via Unreal Engine Marketplace (FAB)
- For immediate support needs, create an issue in the repository
