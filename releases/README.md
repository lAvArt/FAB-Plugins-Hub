# Release Notes

This directory contains release notes and changelogs for all FAB plugins.

## Current Releases

Release notes are organized by plugin. Check the individual plugin directories for detailed changelogs.

## Release Format

Each release note includes:
- **Version Number**: Semantic versioning (e.g., v1.0.0)
- **Release Date**: When the update was published
- **Compatibility**: Supported Unreal Engine versions
- **Changes**: New features, improvements, bug fixes, and breaking changes
- **Known Issues**: Any identified issues in the release

## Getting Updates

All plugin updates are distributed through the Unreal Engine Marketplace (FAB). Updates can be installed via the Epic Games Launcher.

To check for updates:
1. Open Epic Games Launcher
2. Go to your Library
3. Navigate to the Marketplace tab
4. Look for available updates

## Versioning Scheme

We follow [Semantic Versioning](https://semver.org/):
- **Major (X.0.0)**: Breaking changes or significant new features
- **Minor (0.X.0)**: New features, backward compatible
- **Patch (0.0.X)**: Bug fixes and minor improvements

## Subscribing to Updates

To stay informed about new releases:
- Watch this repository on GitHub
- Check the [Announcements](../announcements/README.md) directory
- Review the [Roadmap](../ROADMAP.md) for planned features
